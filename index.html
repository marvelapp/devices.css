10 Pure CSS (Mostly) Flat Mobile Devices

We've created several minimal iPhone, Android, Lumia and iPad devices in pure CSS to showcase the prototypes our users make on Marvel. Grab the code, remix and enjoy! Made with love by Oleg.

Update: Rewritten code & brand new iPhone 6 & iPhone 6 Plus. Just include devices.min.css and grab the code from any of the devices below.

Grab the code

To use, download & include devices.min.css from Github, and paste the code below into your page.

    <div class="top-bar"></div>
    <div class="sleep"></div>
    <div class="volume"></div>
    <div class="camera"></div>
    <div class="sensor"></div>
    <div class="speaker"></div>
    <div class="screen">

    </div>
    <div class="home"></div>
    <div class="bottom-bar"></div>
</div>
